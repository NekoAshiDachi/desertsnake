Submit a URL of an EDGAR credit agreement, and once the page has finished loading, check the results below. This classifier uses n-grams with Naive Bayes to train its model. Under the hood, each document is parsed into its individual discrete sections, allowing the possibility of the model classifying more covenants and parsing out more granular data.

EDGAR Scraper

The code behind the classifier includes a webscraper to go through the hundreds of thousands of EDGAR filings each quarter. It is currently set for Exhibit 4s and 10s, but it can be set for other exhibit types as well. Each filing can be analyzed and saved with all of its metadata and documents.
